COOL Compiler

This project that was undertaken in the ECS652 class involved the development of a compiler for the Cool programming language, with the target output being Java code. The Cool language, standing for "Classroom Object-Oriented Language," is a simple, statically-typed, and education-oriented programming language commonly used in compiler construction courses. The primary objective of this project was to demonstrate an understanding of various compiler design and implementation concepts.
