Image Processor

This Java-based image processing application, "ImageProcessor," provides a comprehensive suite of functionalities for manipulating and analyzing images. It allows users to perform operations such as rescaling, color shifting, noise addition, convolution, histogram equalization, and more through a user-friendly graphical interface. The program supports multiple image operations including arithmetic and bitwise manipulations, filters, and thresholding techniques. Key features include a history of actions for easy undo, real-time adjustments via sliders or input dialogs, and options to save or reload processed images.
